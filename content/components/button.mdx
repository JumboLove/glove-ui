Buttons are enhanced with a larger target area for more reliable clicks.

On long press, a button will show a copy of the button above as a popover.

<Callout variant="warning" className="my-6 max-w-[60ch]">
	Buttons use the experimental [popover](https://caniuse.com/?search=popover)
	and [anchor](https://caniuse.com/?search=anchor) browser APIs. Please use
	caution when implementing as browsers implement these features.
</Callout>
